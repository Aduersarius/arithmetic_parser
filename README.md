# arithmetic_parser

Напишите калькулятор, поддерживающий следующие арфимитические операции:
- a + b
- a - b
- a * b
- a / b
- a % b
- a & b
- a | b
- a ^ b
- a << b
- a >> b

### Входные данные
Во входных данных задана строка, вида `<число> <операция> <число>`

### Выходные данные
Выведите результат операции

### Примеры
#### входные данные
`3 + 4`
#### выходные данные
`7`
#### входные данные
`3 * 2`
#### выходные данные
`6`
